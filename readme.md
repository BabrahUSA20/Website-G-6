<a id="installation"></a>
## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/your-repo.git
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm start
```

<a id="development-workflow"></a>
## Development Workflow
We follow these practices:
- Rotating pair programming sessions
- Daily standups
- Code reviews for all pull requests
- Feature branch workflow
- Regular merging to main branch

<a id="project-structure"></a>
## Project Structure

| Branch      | Component  | Pair Assignments       |
|-------------|------------|------------------------|
| `header`    | Header     | @member1 + @member2    |
| `hero`      | Hero       | @member3 + @member4    |
| `contact`   | Contact    | @member5 + @member1    |
| `footer`    | Footer     | @member2 + @member3    |
| `strategy`  | Strategy   | @member4 + @member5    |
| `privacy`   | Privacy    | Rotating pairs         |

To work on a component:
```bash
git checkout branch-name
# Example: git checkout header
```
